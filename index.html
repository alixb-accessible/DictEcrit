Le microphone ne fonctionne pas ? Voici comment l'autoriser

▼

Méthode 1 - Paramètres du site (la plus simple) :

  1. Cliquez sur les trois points verticaux en haut à droite de Chrome
  2. Allez dans Paramètres
  3. Dans la barre de recherche en haut, tapez "micro" ou "microphone"
  4. Cliquez sur Paramètres du site puis Microphone
  5. Vérifiez que "Les sites peuvent demander à utiliser votre micro" est activé
  6. Regardez si cette page est dans la liste "Non autorisés" et retirez-la si c'est le cas
  7. Rechargez cette page (F5)

Méthode 2 - Depuis la barre d'adresse :

  1. Regardez tout à gauche de la barre d'adresse (où il y a l'URL)
  2. Vous devriez voir une icône (cadenas, tune/réglages, ou symbole de connexion)
  3. Cliquez dessus
  4. Si vous voyez "Microphone" dans la liste, changez-le sur "Autoriser"
  5. Sinon, cliquez sur "Paramètres du site" pour accéder aux réglages complets
  6. Rechargez la page (F5)

  2. Cliquez sur "Effacer les autorisations et fermer"
  3. Rechargez la page (F5 ou Ctrl+R)
  4. Une popup apparaîtra vous demandant l'autorisation, cliquez sur "Autoriser"

  2. Cherchez "Microphone"
  3. Changez le réglage à "Autoriser"
  4. Rechargez la page (Cmd+R)

Astuce : Si rien ne fonctionne, vérifiez que votre microphone est bien branché et allumé, et qu'aucune autre application ne l'utilise déjà.

Astuces :
  • Cliquez sur "Démarrer" ou appuyez sur Entrée pour commencer la dictée
  • Dites "point", "virgule", "nouveau paragraphe" pour la ponctuation
  • Aucune limite de caractères - dictez autant que vous voulez
  • Le texte est sauvegardé automatiquement dans l'historique
  • L'audio est téléchargé automatiquement quand vous arrêtez
  • Navigation complète au clavier (Tab, Entrée, Échap)

Votre vie privée est protégée

DictEcrit respecte votre confidentialité :

  • Tout se passe directement sur votre appareil, rien n'est envoyé sur internet
  • Vos enregistrements audio et vos textes restent sur votre ordinateur ou téléphone
  • Aucune donnée n'est collectée, aucun serveur externe n'est sollicité
  • Vous gardez le contrôle total de vos fichiers et pouvez les supprimer quand vous voulez

C'est vous qui décidez ce que vous faites de vos données, et personne d'autre.
